2. Simple Past
Active –> Subject + v2
Passive -> Object + was/were + v3 + by + subject

  1. He wrote a letter.
    A letter was written by him.
  2. They wrote some letters
    some letters were written by them.

3. Present continuous
Subject + is/are/am + ing
Object + is/are/am + being + v3 + subject

  1. He is writing a letter
    A letter is being written by him.
  2. They are writing some letters
    Some letters are being written by them.

4. Past continuous
Subject + was/were + ing
Object + was/were + being + v3 + by + subject

  1. He was writing a letter
    A letter was being written by him
  2. They were writing some letters
    Some letters were being written by them

5. Present perfect tense
Subject + has/have + v3
Object + has/have + been + v3 + by + subject

  1. He has written a letter
    A letter has been written by him
  2. They have written some letters
    Some letters have been written by them

6. Future tense
Subject + will/shall + base verb
Object + will/shall + be + v3 + by + subject
Note: I/we shall is used. For all others will is used

  1. He will write a letter
    A letter will be written by him
  2. They will write some letters
    Some letters will be written by them
  3. I shall write a letter
    A letter will be written by me

Order/ Request
Request -> Let + object + be + v3

Open the door
Let the door be opened

1. A question beginning with (question word) + do/does/did
Passive-> (question word) + is/are/am/was/were + object + v3 + by + subject + ?
Note: if do/does -> is/are/am. If did -> was/were

  • Eg: How did you complete the work?
    How was the work completed by you?
  • Do you finish your homework?
    Is your homework finished by you?

2. A question beginning with have/has/had/will/would/shall/should/may/might/can/could.
Passive-> have/has/had/will/would/shall/should/may/might/can/could + object + be/been v3 + by + subject + ?
have/has/had -> been. Others use -> be

  • Have you seen her?
    has she been seen by you?
  • Shall I follow you?
    will you be followed by me?

3. A sentence beginning with – he says that, he said that, he suggest that 
Passive-> it + is/was/has been/ will be + v3. 
Note: When there is no object in the remaining part, use the remaining part without any change. When there is object in the remaining part write it in appropriate tense.

  • He says that he works hard
    It is said that he works hard
  • He said that he had met his friend
    It was said that his friend had been met by him.
